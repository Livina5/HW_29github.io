Java Script

Способи використання JS:

  • Динамічне взаємодія з користувачем;
  • Асиметрична обробка запитів (AJAX і Fetch API);
  • Робота з даними;
  • Робота з анімаціями та ефектами;
  • Інтерфейси для користувача (UI/UX);
  • Робота з веб-сервісами;
  • Інтерфейси для користувача (UI/UX);
  • Node.js та серверна сторона;
  • Мобільні додатки та десктопні додатки;
  • Ігри;
  • Типізація та модульність коду;
  • Обробка асинхронних операцій;
  • Автоматизація завдань;
  • Плагіни та розширення для браузерів.

Це лише частина можливих способів використання JavaScript, оскільки мова дуже гнучка і застосовується в різних сферах програмування.

Способи написання імен змінних:

  • camelCase (Верблюжий стиль);
  • PascalCase;
  • snake_case.
